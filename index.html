Quick Guide

What's the difference between Linear and Radial gradients?

Linear gradients transition colors in a straight line, while radial gradients transition from a center point outward in a circular pattern.

How do I use CSS gradients in my website?

Copy the generated CSS and apply it to any element's background property. Works in all modern browsers.

Can I add more than 2 colors?

Yes! Click the "+" button to add up to 6 color stops and create complex multi-color gradients.
